# Rethinking Loss Functions for Fact Verification


A Docker environment for [thunlp/KernelGAT](https://github.com/thunlp/KernelGAT)


## Build a docker image

```
make docker-build
```


## Download data and checkpoint
```
make download
```


## Preprocess
```
make prepro
```


## Usage

### Train
```
make kgat
```

### Test
```
make test
```

### Evaluate
```
make eval
```
